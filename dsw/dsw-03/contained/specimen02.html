What is a design token?

A design token is a named design decision stored as a CSS custom property. Tokens live in @layer tokens and every other layer references them by name.

Why OKLCH for color?

OKLCH provides perceptual uniformity — equal numeric steps look visually equal. This makes building harmonious, accessible color scales much more predictable than HSL or sRGB.

What is the cascade layer order?

reset → tokens → elements → layout → objects → qualities → roles → states → utilities. Later layers win over earlier ones for the same specificity.
