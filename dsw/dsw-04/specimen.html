What is a design token?

Design tokens are named entities that store visual design attributes. They are the single source of truth for values like colour, spacing, and typography across a design system.

How does the cascade layering work?

The @layer declaration at the top of the file establishes explicit layer priority. Later layers win over earlier ones when specificity is equal, making the order predictable without resorting to !important.

Why OKLCH for color?

OKLCH is a perceptually uniform color space. When you set a lightness value, it actually looks that light to the human eye — unlike HSL which has wildly different perceived brightness at the same L value depending on hue.
